<h3>Como rodar o projeto</h3>

```bash
# Clonar repositório
$ git clone https://github.com/stafussa/lavajato-electron-database

# Entrar no repositório
$ cd lavajato-electron-database

# Instalar dependências
$ npm install

# Iniciar aplicação
$ npm start

```
